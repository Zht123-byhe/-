# 校园学生信息管理系统

#### 介绍
项目简介 
• 开发语言：Python 3.x 
• 技术栈：Tkinter（GUI 图形界面）+ SQLite（本地数据库） 
• 项目类型：桌面端管理系统 
• 适用场景：课程实践项目、自主开发软件、实训作品 
• 核心特点：界面简洁、功能完整、无需联网、本地运行、代码规范易读

#### 使用说明

1. 确保电脑安装了 Python 3.x（自带 Tkinter，无需额外安装库） 
2. 将代码保存为 student_management.py 
3. 双击运行文件，自动弹出图形化界面 
4. 系统会自动创建本地数据库，所有数据保存在 student.db 文件中
